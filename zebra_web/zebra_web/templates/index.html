About our projects

We’re implementing our version of a Zero Effort Bilateral Re-occurring Authentication method using wearable tech, more specifically with an Apple Watch. Users will log into this web portal and their actions will be monitored through the Apple Watch. When either the user has not interacted with the system log enough, or steps away for a moment they will be logged out of the system. Also, if another user tries to use a login one another user’s behalf they will be logged out.
